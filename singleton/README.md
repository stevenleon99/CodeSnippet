# Singleton Pattern

C++ singleton design pattern implementation.

## Contents

- **singleton.cpp** - Singleton pattern implementation
- **.vscode/** - VS Code configuration

## Building

```bash
# Compile using VS Code task or directly
g++.exe -fdiagnostics-color=always -g singleton.cpp -o singleton.exe
```

## Pattern

Demonstrates the singleton pattern for ensuring a class has only one instance with global access.
