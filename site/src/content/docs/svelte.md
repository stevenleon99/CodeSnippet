Svelte web application project built with the [`sv`](https://github.com/sveltejs/cli) CLI.

## Developing

```bash
npm install
npm run dev

# or start the server and open in a new browser tab
npm run dev -- --open
```

## Building

```bash
npm run build
```

Preview the production build with `npm run preview`.

> To deploy, you may need to install an [adapter](https://svelte.dev/docs/kit/adapters) for your target environment.
