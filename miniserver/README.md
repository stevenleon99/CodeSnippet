# Mini Server

Lightweight server and launcher application examples.

## Contents

- **call_lancher_agent.ps1** - PowerShell launcher script
- **gui_launcher_agent.py** - Python GUI launcher
- **memcheck.py** - Memory checking utility
- **__pycache__/** - Python bytecode cache

## Usage

```bash
# Run Python GUI launcher
python gui_launcher_agent.py

# Run PowerShell launcher
powershell -ExecutionPolicy Bypass -File call_lancher_agent.ps1
```
