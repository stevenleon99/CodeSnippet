# CMake Configuration & Utilities

CMake build system configuration files and utility functions for the project.

## Contents

- **CMakeLists.txt** - Main CMake configuration file
- **cmake_util.cmake** - Reusable CMake utility functions
- **build.txt** - Build system documentation
- **addcustomtarget.txt** - Custom CMake target patterns

## Key Features

- Cross-platform build configuration
- Utility functions for common build tasks
- Custom target generation patterns

## Usage

```bash
# Configure and build
cmake -B build -S .
cmake --build build
```
