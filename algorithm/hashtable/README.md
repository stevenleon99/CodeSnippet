# Hash Table Algorithms

Hash-based data structures and algorithms.

## Contents

- **hashtable.cpp** - Hash table implementation
- **hashmap_3sum.cpp** - 3-sum problem using hash maps
- **unorderset.cpp** - Unordered set patterns

## Building

```bash
g++.exe -fdiagnostics-color=always -g hashtable.cpp -o hashtable.exe
```

## Pattern

Demonstrates efficient O(1) average-case lookups and deduplication using hash tables.
