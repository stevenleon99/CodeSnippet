# Algorithm Examples

Comprehensive collection of algorithm implementations covering arrays, dynamic programming, hash tables, linked lists, and graph algorithms.

## Directory Structure

- **array/** - Array manipulation and searching algorithms
- **dynamicprogram/** - Dynamic programming solutions
- **hashtable/** - Hash-based data structures
- **linkednode/** - Linked list implementations
- **topoprogram/** - Topological and graph algorithms

## Root-Level Files

- **disjointSetUnion.cpp** - Union-Find data structure
- **KMP.cpp** - Knuth-Morris-Pratt string searching algorithm
- **priorityqueue.cpp** - Priority queue implementation
- **dataAnnotate.py** - Data annotation utilities

## Building

All C++ files compile with:
```bash
g++.exe -fdiagnostics-color=always -g <filename> -o <filename>.exe
```
