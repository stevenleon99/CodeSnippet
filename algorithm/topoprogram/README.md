# Topological & Graph Algorithms

Graph traversal, shortest path, and topological sorting algorithms.

## Contents

- **dijkstra.cpp** - Dijkstra's shortest path algorithm
- **bellmanford.cpp** - Bellman-Ford algorithm (handles negative weights)
- **prim.cpp** - Prim's minimum spanning tree algorithm
- **numberIsland.cpp** - Island counting problem
- **largestIsland.cpp** - Largest island area problem
- **islandPerimeter.cpp** - Island perimeter calculation
- **buildlargestisland.cpp** - Build largest connected component
- **reachablepath.cpp** - Reachability analysis
- **searchnjoinset.cpp** - Search and join set operations

## Building

```bash
g++.exe -fdiagnostics-color=always -g dijkstra.cpp -o dijkstra.exe
```

## Algorithms

- **Shortest Path**: Dijkstra, Bellman-Ford
- **Minimum Spanning Tree**: Prim's algorithm
- **Connected Components**: Island counting and analysis
- **Graph Traversal**: DFS/BFS patterns
