# Dynamic Programming Algorithms

Dynamic programming solutions to optimization problems.

## Contents

- **01backpack.cpp** - 0/1 knapsack problem
- **climbstairs.cpp** - Staircase climbing problem
- **completepack.cpp** - Complete knapsack problem

## Building

```bash
g++.exe -fdiagnostics-color=always -g 01backpack.cpp -o backpack.exe
```

## Pattern

These examples demonstrate:
- Memoization and tabulation approaches
- State space definition
- Recurrence relations
