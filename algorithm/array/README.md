# Array Algorithms

Array manipulation and searching algorithms.

## Contents

- **binarysearch.cpp** - Binary search implementation
- **minimalsubarray.cpp** - Minimal subarray finding
- **removeelement.cpp** - Element removal techniques
- **rotmatrix.cpp** - Matrix rotation algorithms
- **squarearray.cpp** - Square array operations

## Building

```bash
# Compile individual files
g++.exe -fdiagnostics-color=always -g binarysearch.cpp -o binarysearch.exe
```
